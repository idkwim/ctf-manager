CTF Management System

Simple CTF management system for #overflowsec.

If you're a part of the overflowsec CTF team and landed here to get details about an ongoing or nearly arriving CTF event, please message an OP and have them send you the CTF event code.

Paste the CTF event code and you will be taken to the team CTF management page. This page will contain CTF event team login credentials as well as start and end times. You will be able to claim individual event tasks via this system so everyone knows who's working on what.
